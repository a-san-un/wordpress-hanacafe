## §1 管理画面チェック（WordPress にログインして確認）

| # | 項目 | 場所 |
|---|------|------|
| 1-1 | WP・プラグインが最新か | ダッシュボード → 更新 |
| 1-2 | 「検索エンジンにインデックスさせない」が **OFF** か | 設定 → 表示設定 |
| 1-3 | ACF マスターデータが正しく入力されているか | 各固定ページ（common-info / access-info 等） |
| 1-4 | Yoast SEO が有効・OGP 画像が設定されているか | プラグイン / Yoast → ソーシャル |
| 1-5 | デバッグモードが **OFF** か | wp-config.php の `WP_DEBUG` |

### 主要プラグイン確認メモ

| プラグイン | 用途 | 注意 |
|-----------|------|------|
| CPT UI | カスタム投稿タイプ管理 | 更新後にメニュー CPT の動作確認 |
| All-in-One WP Migration | バックアップ・移行 | ⚠️ 更新後は必ずバックアップ取得・復元テストを実施 |

## §3 コマンドチェック（Chrome DevTools Console で実行）

> URL のパス部分は環境に応じて変更して使う。

### 3-1 SEO・メタタグ総合診断

チェックしたいページを開いて実行。結果に `❌` がなければOK。

```js
const seo = {
  title:        document.title,
  description:  document.querySelector('meta[name="description"]')?.content ?? '❌',
  canonical:    document.querySelector('link[rel="canonical"]')?.href ?? '❌',
  robots:       document.querySelector('meta[name="robots"]')?.content ?? '❌',
  ogTitle:      document.querySelector('meta[property="og:title"]')?.content ?? '❌',
  ogImage:      document.querySelector('meta[property="og:image"]')?.content ?? '❌',
  twitterCard:  document.querySelector('meta[name="twitter:card"]')?.content ?? '❌',
  h1Count:      document.querySelectorAll('h1').length,
  h2Count:      document.querySelectorAll('h2').length,
  imgTotal:     document.querySelectorAll('img').length,
  imgNoAlt:     [...document.querySelectorAll('img')].filter(i => !i.alt).length,
  viewport:     document.querySelector('meta[name="viewport"]')?.content ?? '❌',
};
console.table(seo);
```

**期待値:** 全項目に値あり・`h1Count: 1`・`imgNoAlt: 0`

### 3-4 noindex 確認（マスターページ一括）

URL は本番環境に合わせて変更すること。

```js
const pages = [
  '/common-info/',
  '/access-info/',
  '/menu-info/',
  '/about-seats/',
  '/news-info/',
];
Promise.all(pages.map(url =>
  fetch(url).then(r => r.text()).then(html => {
    const m = html.match(/name=['"]robots['"][^>]*content=['"]([^'"]+)/i);
    return `${url}: ${m ? m[1] : '❌ robots なし'}`;
  })
)).then(r => console.log(r.join('\n')));
```

**期待値:** 全ページ `noindex, nofollow`
