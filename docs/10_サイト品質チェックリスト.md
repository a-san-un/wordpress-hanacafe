## §0 実行環境・前提条件

| 条件           | 指定                                                                                        |
| ------------ | ----------------------------------------------------------------------------------------- |
| **実行ブラウザ**   | Google Chrome（最新版）                                                                        |
| **実行ページ**    | TOPページ（本番: `https://intp.site/2999/WordPress/hanacafe/` / ローカル: `http://hanacafe.local/`） |
| **ログイン状態**   | **WordPress管理画面からログアウトした状態**で実行すること（ログイン中は管理バーのGravatarが `imgNoAlt` に混入するため）              |
| **デバイス幅**    | DevTools を閉じた状態のPC表示（幎1280px以上）で §1〜§2 を確認後、DevTools で幎375pxに切り替えてSP表示も目視確認               |
| **DevTools** | Chrome DevTools → Console タブで §3 のコマンドを実行                                                 |

> §3 のコマンドはすべて **環境自動判別**（`location.hostname` で本番/ローカルを切り替え）のため、どちらの環境でもそのまま貼り付けて実行できます。

## §1 管理画面チェック（WordPress にログインして確認）

| #   | 項目                            | 場所                                                                                        |
| --- | ----------------------------- | ----------------------------------------------------------------------------------------- |
| 1-1 | WP・プラグインが最新か                  | ダッシュボード → 更新                                                                              |
| 1-2 | 「検索エンジンにインデックスさせない」が OFF か    | WordPress設定 → 表示設定                                                                        |
| 1-3 | ACF マスターデータが正しく入力されているか       | 各固定ページ（common-info / access-info 等）                                                       |
| 1-4 | Yoast SEO が有効・OGP 画像が設定されているか | Yoast SEO → 設定 → コンテンツタイプ → ホームページリンク → 固定ページ「Home」編集画面 → Yoastメタボックス → ソーシャルタブ → ソーシャル画像 |
| 1-5 | デバッグモードが OFF か                | wp-config.php の WP\_DEBUG                                                                 |

### 主要プラグイン確認メモ

| プラグイン                   | 用途                | 注意                         |
| ----------------------- | ----------------- | -------------------------- |
| ACF                     | カスタムフィールド管理       | 更新後に各マスターページのデータ表示を確認      |
| Yoast SEO               | SEO・XMLサイトマップ・OGP | 更新後にサイトマップ・noindex設定を再確認   |
| CPT UI                  | カスタム投稿タイプ管理       | 更新後にメニュー CPT の動作確認         |
| All-in-One WP Migration | バックアップ・移行         | ⚠️ 更新後は必ずバックアップ取得・復元テストを実施 |
| WP Super Cache          | キャッシュ             | 更新後にキャッシュをクリアして表示確認        |

### 3-1 SEO・メタタグ総合診断

```js
const seo = {
  title:        document.title,
  description:  document.querySelector('meta[name="description"]')?.content ?? '❌',
  canonical:    document.querySelector('link[rel="canonical"]')?.href ?? '❌',
  robots:       document.querySelector('meta[name="robots"]')?.content ?? '❌',
  ogTitle:      document.querySelector('meta[property="og:title"]')?.content ?? '❌',
  ogImage:      document.querySelector('meta[property="og:image"]')?.content ?? '❌',
  twitterCard:  document.querySelector('meta[name="twitter:card"]')?.content ?? '❌',
  h1Count:      document.querySelectorAll('h1').length,
  h2Count:      document.querySelectorAll('h2').length,
  imgTotal:     document.querySelectorAll('img').length,
  imgNoAlt:     [...document.querySelectorAll('img')].filter(i => !i.alt).length,
  viewport:     document.querySelector('meta[name="viewport"]')?.content ?? '❌',
};
console.table(seo);
```

**期待値:** 全項目に値あり・`h1Count: 1`・`imgNoAlt: 0` （⚠️ ログイン中は管理バーのGravatarが混入するため、必ずログアウト状態で確認）

### 3-4 noindex 確認（マスターページ一括）

```js
const base = location.hostname === 'hanacafe.local'
  ? ''
  : '/2999/WordPress/hanacafe';

const pages = [
  base + '/common-info/',
  base + '/access-info/',
  base + '/menu-info/',
  base + '/about-seats/',
  base + '/news-info/',
];
Promise.all(pages.map(url =>
  fetch(url).then(r => r.text()).then(html => {
    const m = html.match(/name=['"']robots['"'][^>]*content=['"']([^'"']+)/i);
    return `${url}: ${m ? m[1] : '❌ robots なし'}`;
  })
)).then(r => console.log(r.join('\n')));
```

**期待値:** 全ページ `noindex, nofollow`
