## 1. 環境・WordPress設定

| # | チェック項目 | 確認方法 |
|---|---|---|
| 1-1 | WordPress のバージョンが最新か | 管理画面「ダッシュボード」→「更新」 |
| 1-2 | プラグイン（ACF / CPT UI 等）が最新か | 管理画面「プラグイン」→「更新」 |
| 1-3 | パーマリンク設定が「投稿名」になっているか | 設定 → パーマリンク → 「投稿名」を選択 → 保存 |
| 1-4 | 「検索エンジンがサイトをインデックスしないようにする」が **OFF** になっているか | 設定 → 表示設定 → チェックボックスを確認 |
| 1-5 | サイトURL・WordPressURLが本番ドメインになっているか | 設定 → 一般 |
| 1-6 | 管理者メールアドレスが正しいか | 設定 → 一般 |
| 1-7 | デフォルトのサンプルページ・投稿が削除されているか | 固定ページ・投稿一覧を確認 |

## 7. DevTools 診断コマンド集

> すべて Chrome DevTools の **Console** タブ**で実行**。
> URLのパス部分は環境に応じて適宜変更して使ってください。

### 7-1 SEO 総合診断（チェックしたいページで実行）

メタタグ・ OGP ・見出し・画像 alt を一括確認。結果に `❌ なし` がなければ公開失格項目なし。

```js
const seo = {
  title:          document.title,
  description:    document.querySelector('meta[name="description"]')?.content ?? '❌ なし',
  canonical:      document.querySelector('link[rel="canonical"]')?.href ?? '❌ なし',
  robots:         document.querySelector('meta[name="robots"]')?.content ?? '❌ なし',
  ogTitle:        document.querySelector('meta[property="og:title"]')?.content ?? '❌ なし',
  ogDescription:  document.querySelector('meta[property="og:description"]')?.content ?? '❌ なし',
  ogImage:        document.querySelector('meta[property="og:image"]')?.content ?? '❌ なし',
  ogUrl:          document.querySelector('meta[property="og:url"]')?.content ?? '❌ なし',
  ogType:         document.querySelector('meta[property="og:type"]')?.content ?? '❌ なし',
  twitterCard:    document.querySelector('meta[name="twitter:card"]')?.content ?? '❌ なし',
  h1Count:        document.querySelectorAll('h1').length,
  h1Text:         document.querySelector('h1')?.textContent?.trim().slice(0, 40) ?? '❌ なし',
  h2Count:        document.querySelectorAll('h2').length,
  imgTotal:       document.querySelectorAll('img').length,
  imgNoAlt:       [...document.querySelectorAll('img')].filter(i => !i.alt).length,
  viewport:       document.querySelector('meta[name="viewport"]')?.content ?? '❌ なし',
};
console.table(seo);
```

**期待値:** 全項目に値が入り、`h1Count: 1`、`imgNoAlt: 0`（管理バーの gravatar 画像は除く）。

---

### 7-2 noindex 複数ページ一括確認

マスター5ページが意図通り noindex になっているか確認。**URL リストは環境に応じて変更**。

```js
const pages = [
  '/2999/WordPress/hanacafe/common-info/',
  '/2999/WordPress/hanacafe/access-info/',
  '/2999/WordPress/hanacafe/menu-info/',
  '/2999/WordPress/hanacafe/about-seats/',
  '/2999/WordPress/hanacafe/news-info/',
];

Promise.all(pages.map(url =>
  fetch(url).then(r => r.text()).then(html => {
    const m = html.match(/<meta name=['"]robots['"][^>]*content=['"]([^'"]+)['"]/i);
    return `${url.split('/').at(-2)}: ${m ? m[1] : '❌ robots meta なし'}`;
  })
)).then(r => console.log(r.join('\n')));
```

**期待値:** 全ページ `noindex, nofollow`。

### 7-3 見出し階層詳細確認

```js
['h1','h2','h3'].forEach(tag => {
  const els = [...document.querySelectorAll(tag)];
  console.log(`=== ${tag} (${els.length}件) ===`);
  els.forEach(el => console.log(el.textContent.trim()));
});
```

**期待値:** `h1: 1件`、`h2: 複数`、`h3: 複数`。

### 7-4 alt 抗け画像の特定

`imgNoAlt` が 0 より大きい場合に実行。

```js
[...document.querySelectorAll('img')]
  .filter(i => !i.alt)
  .forEach(i => console.log(i.src));
```

**注意:** `secure.gravatar.com` の画像は WordPress 管理バーのアバター。**ログイン中のみ表示されるため問題なし**。
