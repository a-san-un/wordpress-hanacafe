## 1. 環境・WordPress設定

| # | チェック項目 | 確認方法 |
|---|---|---|
| 1-1 | WordPress のバージョンが最新か | 管理画面「ダッシュボード」→「更新」 |
| 1-2 | プラグイン（ACF / CPT UI 等）が最新か | 管理画面「プラグイン」→「更新」 |
| 1-3 | パーマリンク設定が「投稿名」になっているか | 設定 → パーマリンク → 「投稿名」を選択 → 保存 |
| 1-4 | 「検索エンジンがサイトをインデックスしないようにする」が **OFF** になっているか | 設定 → 表示設定 → チェックボックスを確認 |
| 1-5 | サイトURL・WordPressURLが本番ドメインになっているか | 設定 → 一般 |
| 1-6 | 管理者メールアドレスが正しいか | 設定 → 一般 |
| 1-7 | デフォルトのサンプルページ・投稿が削除されているか | 固定ページ・投稿一覧を確認 |

## 4. セキュリティ確認

| # | チェック項目 | 確認方法 |
|---|---|---|
| 4-1 | WordPress のデバッグモードが **OFF** になっているか | `wp-config.php` の `WP_DEBUG` が `false` |
| 4-2 | 管理者パスワードが十分に強いか（辞書攻撃耐性） | WordPress ユーザー設定で確認 |
| 4-3 | 不要なユーザーアカウントが削除されているか | 管理画面「ユーザー」一覧 |
| 4-4 | ファイルパーミッションが適切か（`wp-config.php` は 400） | サーバー設定・FTPで確認 |
| 4-5 | XML-RPC が不要な場合は無効化されているか | プラグイン等で確認 |

## 5. SEO・アクセシビリティ確認

| # | チェック項目 | 確認方法 |
|---|---|---|
| 5-1 | 各ページのタイトルタグが適切に設定されているか | ブラウザタブ・ソース確認 |
| 5-2 | OGP（og:title / og:image）が設定されているか | ソース or SNSデバッガーで確認 |
| 5-3 | 全画像に `alt` 属性が設定されているか（自動補完機能の動作確認） | Chrome DevTools |
| 5-4 | メニュー投稿・席情報カードの `alt` が空でないか | 各ページを目視確認 |
| 5-5 | 見出し階層（h1→h2→h3）が論理的に構成されているか | Chrome DevTools |
| 5-6 | キーボード操作でナビゲーションが操作できるか | Tab キーで確認 |
| 5-7 | Yoast SEO がインストール・有効化されているか | 管理画面「プラグイン」 |
| 5-8 | XMLサイトマップが生成されているか（/sitemap_index.xml） | ブラウザで直接アクセス |
| 5-9 | OGP画像がYoast SEOで設定されているか | Yoast SEO → ソーシャル |
| 5-10 | 投稿者アーカイブが無効化されているか | Yoast SEO → 詳細設定 |

## 7. 本番移行手順（直接アップロード方式）

> ℹ️ 本プロジェクトは All-in-One WP Migration を使わず、FileZilla で直接サーバーにアップロードする方式で移行する。

### 事前準備（ローカル）

| # | 作業 |
|---|---|
| 7-1 | FileZilla で WordPress ファイルを本番サーバー `/hanacafe/wp/` にアップロード |
| 7-2 | `/hanacafe/index.php` を配置（内容: `require __DIR__ . '/wp/wp-blog-header.php';`） |
| 7-3 | `wp-config.php` に `WP_HOME` / `WP_SITEURL` を追記 |

### 本番サーバー作業

| # | 作業 |
|---|---|
| 7-4 | 管理画面 → 設定 → 一般 でサイトアドレスを `/wp` なしに変更 |
| 7-5 | `wp-config.php` のパーミッションを 400 に変更 |
| 7-6 | `wp/.htaccess` に `wp-config.php` へのアクセス拒否コードを追記 |

### アップロード後の確認

| # | 作業 |
|---|---|
| 7-7 | 設定 → 一般 で WordPress アドレス・サイトアドレスが本番URLになっているか確認 |
| 7-8 | 設定 → 表示設定 で「検索エンジンにインデックスさせない」が **OFF** か確認（1-4） |
| 7-9 | `wp/wp-config.php` の `WP_DEBUG` が `false` になっているか確認（4-1） |
| 7-10 | 本番URLでトップページ・各ページが正常に表示されるか確認（セクション3） |
| 7-11 | パンくずリンクのURLに `/wp/` が混入していないか確認（サブディレクトリ構成時に発生しやすい） |
| 7-12 | SNSシェア時にOGP画像・タイトルが正しく表示されるか |
| 7-13 | お問い合わせ・予約リンクが正しく遷移するか |
| 7-14 | Google Search Console にサイトマップを送信したか |
